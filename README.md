# weblog-django
the weblog has two apps :
* blog app
* authentication app

blog app contains posts and post templates

authentication app contains auth operations like signup , login and logout

# requirements

* asgiref==3.6.0

* django-autoslug==1.9.9
* django-js-asset==2.0.0
* Pillow==9.4.0
* sqlparse==0.4.3
